
# 🧰 Toolsyde - API

A RESTful API built with Spring Boot for Toolsyde. It follows a microservices architecture and serves as the backend for organizing the developer utilities of the application.

## ⚙️ Features

- Microservices structure.
- MVC Arquitecture.
- RESTful API.
